```go
func numTrees(n int) int {
   dp := make([]int, n+1)
   dp[0], dp[1] = 1, 1
   for i := 2; i <= n; i++ {
      for j := 1; j <= i; j++ {
         dp[i] += dp[j-1] * dp[i-j]
      }
   }
   return dp[n]
}
```

一道不简单的题，看了解题思路还想了很久，当前理解：dp[i]会等于所有j等于1到i的数作为根节点不同情况之和

每个数字j的情况会等于左节点的所有情况乘上有节点的所有情况以此递推到dp[0]和dp[1]，如此反复；

这边为什么是乘呢：两边的不同情况的组合当然是乘起来
